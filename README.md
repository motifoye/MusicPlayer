# MusicPlayer

Музыкальный плеер - это мой первый проект. Не просто тренировка, отдельный кусок кода, а полноценное приложение.

Изначально задумывалось разработать приложение, которое могло бы воспроизводить музыку из ВКонтакте. Но для начала, я решил сделать простой плеер, который будет воспроизводить музыку с компьютера пользователя.

В дальнейшем проект будет обновляться, не могу обещать, что это будет в ближайшее время. Появится возможность авторизации через вконтакте и воспроизведение музыки со страницы пользователя.

Проект разрабатывается следуя паттерну MVVM. Внем используются различные вспомогательные библиотеки (DevExpressMvvm, Fody, PropertyChanged.Fody и другие). Так же в планах сократить количество сторонних библиотек.
